Slop Index

On-page slop detector: clichés, hedges, repetition, sentence bloat. Client-side, fast, ruthless.

GitHub Repo Live Demo (HF) Write-up VibeAxis

What it flags

  • “We believe…”, “Unlock…”, “Leverage…”—empty calories.
  • Hedges (“might”, “could”, “appears to”).
  • Repetition loops + sentence bloat.

Privacy note: the lite demo is client-side only. No uploads, no storage.

Use it

Install from the repo, or try the Slop Index Lite demo and paste any URL or text. Shipping use? Embed the lib and run checks on content before publish.

Integrations welcome—open an issue with your CMS/environment.
